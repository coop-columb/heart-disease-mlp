# Validation Scripts

This directory contains scripts for validating the ML pipeline and API functionality.

## Structure

```
validation/
├── ml/          # ML pipeline validation
├── api/         # API validation
└── security/    # Security checks
```

## Usage

All scripts support:
- Configurable thresholds
- CI/CD integration
- Local development use
- Detailed reporting
